# ceil()

## Description

Calculates the closest int value that is greater than or equal to the value of the parameter. For example, `ceil(9.03)` returns the value 10.

## Examples

```py
x = 8.22  
a = ceil(x)  # Sets 'a' to 9
```

## Syntax

ceil(n)

## Parameters

| Inputs | Description |
|--------|-------------|
| n (float) | number to compute |

## Return

int
