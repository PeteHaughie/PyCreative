# floor()

## Description

Calculates the closest int value that is less than or equal to the value of the parameter.

## Examples

```py
x = 2.88
a = floor(x) # Sets 'a' to 2
```

## Syntax

floor(n)

## Parameters

| Inputs | Description |
|--------|-------------|
| n (float) | number to round-down |

## Return

int
