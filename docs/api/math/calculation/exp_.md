# exp()

## Description

Returns Euler's number e (2.71828...) raised to the power of the value parameter.

## Examples

```py
v1 = self.exp(1.0)
self.println(v1)  # Prints "2.7182817"
```

## Syntax

exp(n)

## Parameters

| Inputs | Description |
|--------|-------------|
| n (float) | the exponent to raise |

## Return

float
