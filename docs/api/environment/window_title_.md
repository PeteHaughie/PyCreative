# window_title()

## Description

Set the title of the sketch window.

## Example

```py
def setup():
    size(400, 400)
    window_title("My Sketch")  # Set the window title
```

## Syntax

window_title(title)

## Parameters

| Input | Description |
|-------|-------------|
| title (str) | The title to set for the window |

## Return

pass
